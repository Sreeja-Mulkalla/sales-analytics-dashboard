# Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data using MariaDB and visualizes business insights through an interactive dashboard built in Looker Studio.

## Tools Used

- MariaDB
- SQL
- Google Sheets
- Looker Studio

## Dataset

Sample Superstore Dataset

- Total Records: 9,994
- Fields include Sales, Profit, Quantity, Category, Region, State, and Discount.

## Business Questions Answered

- What is the total revenue generated?
- What is the total profit earned?
- Which region generates the highest sales?
- Which product category is most profitable?
- How do discounts affect profitability?
- Which sub-categories perform best and worst?

## Key Insights

- Total Sales: $2.3 Million
- Total Profit: $286.4 Thousand
- Total Quantity Sold: 37.9 Thousand Units
- West region generated the highest sales.
- Technology was the most profitable category.
- Tables and Bookcases were among the least profitable sub-categories.
- Higher discounts showed a negative impact on profitability.
